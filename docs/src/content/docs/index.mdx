## Why CrumbleCracker?

CrumbleCracker runs OCI containers inside lightweight virtual machines, combining the convenience of container images with the security of hardware virtualization.

## Quick Example

```go
package main

import (
    "fmt"
    cc "github.com/tinyrange/cc"
)

func main() {
    // Create OCI client and pull an image
    client, _ := cc.NewOCIClient()
    source, _ := client.Pull(ctx, "alpine:latest")

    // Start a VM from the image
    instance, _ := cc.New(source, cc.WithMemoryMB(256))
    defer instance.Close()

    // Run a command (mirrors os/exec)
    output, _ := instance.Command("echo", "Hello from VM!").Output()
    fmt.Println(string(output))
}
```
