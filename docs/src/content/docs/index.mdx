## Why CrumbleCracker?

CrumbleCracker runs OCI containers inside lightweight virtual machines, combining the convenience of container images with the security of hardware virtualization.

<CardGrid stagger>
	<Card title="Container Compatible" icon="seti:docker">
		Pull any Docker or OCI image and run it directly. No conversion needed.
	</Card>
	<Card title="Hardware Isolation" icon="laptop">
		Each VM runs with its own kernel, providing true isolation via your platform's native hypervisor.
	</Card>
	<Card title="Go-Native API" icon="seti:go">
		Program VMs using familiar Go patterns. The API mirrors `os`, `os/exec`, and `net` packages.
	</Card>
	<Card title="Cross-Platform" icon="linux">
		Runs on macOS (Hypervisor.framework), Linux (KVM), and Windows (WHP).
	</Card>
</CardGrid>

## Quick Example

```go
package main

import (
    "fmt"
    cc "github.com/tinyrange/cc"
)

func main() {
    // Create OCI client and pull an image
    client, _ := cc.NewOCIClient()
    source, _ := client.Pull(ctx, "alpine:latest")

    // Start a VM from the image
    instance, _ := cc.New(source, cc.WithMemoryMB(256))
    defer instance.Close()

    // Run a command (mirrors os/exec)
    output, _ := instance.Command("echo", "Hello from VM!").Output()
    fmt.Println(string(output))
}
```
