# pyneurodesk

`pyneurodesk` is a small Python client for the `ccx3` HTTP API, with an initial
focus on Neurodesk-oriented image import and CVMFS discovery.

Current scope:

- import bootable images with structured source definitions
- list directories in CVMFS
- read files from CVMFS

Planned scope:

- Neurodesk container discovery helpers
- VM lifecycle and exec helpers
- higher-level convenience wrappers for common workflows
